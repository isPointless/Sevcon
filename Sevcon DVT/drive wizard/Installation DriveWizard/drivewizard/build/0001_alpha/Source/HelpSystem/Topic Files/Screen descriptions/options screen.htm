    • Drive Wizard uses CAN bus and Sevcon controller login parameters when connecting to a vehicle. Instead of requireing you to re-enter this infromation every time, Drive Wizard stores this infromation in a Vehicle Profile.
    You can create a Vehicle Profile for every truck type that you need to connect to. Then , next time you run Drive Wizard you can choose yourr Vehicle Profile profile from the drop down list in the Select Profiles Screen

    Modifying the list of available Vehicle Profiles

    Add new Profile

    When you Click this button Drive Wizard creates a new Vehicle profile with default values.

    Select Vehicle Profile

    The highlighted Profile name is the one currently being used by Drive Wizard. Use this control to switch between your Vehicle Profiles to edit or test them.
    If you tick the checkbox next to a profile then drive Wizard will display this as your default profile in the Select Profiles Screen next time you run Drive Wizard

    Delete Selected Profile

    This control allows to to get rid of Vehicle Profiles that you no longer need. When you Click this button Drive Wizard will ask you to confirm that you want to remove the profile highlighted in Vehicle Profile list.

    Deleting a Vehicle Profile is permanent

    Modifying the selected profile: user settings

    Naming your Vehicle Profiles

    When you first create a Vehicle Profile it will be given a name like "new profile0". You should replace this by entering a name here that describes your vehicle type and then press return.
    Drive Wizard will automatically update the Vehicle Profile name and the name of the disk file for this Vehicle Profile (see Copying and Emailing Vehicle Profiles). You can use this control to change a Vehicle Profile at any time

    Modifying the Selected Profile: Connection settings

    Baud Rate Controls

    Select the baud rate of the vehicle from the drop down list.

    'If you do not know the baud rate then select 'autodetect baud'. Drive Wizard will monitor the can bus for a few seconds to detect any CAN devices transmitting heartbeat messages.

    If Drive Wizard is unable to auto detect the baud rate e.g. if the connected devices do not transmit heartbeats, then you will have to try connecting at each possible baud rate

    CAN devices register an error whenever a CAN node attempts to communicate at the wrong baud rate. CAN nodes registering high numbers of errors will go 'off bus. If you have tested at more than one baud rate then , once you have found the correct baud rate, you should cycle vehicle power to clear these errors and then re-connect using this baud rate.

    When first connecting a Vehicle profile you should leave "Search for all connected nodes at this Baud" checked. Once you have successfully connected you should uncheck this to speed up reconnection ( drive Wizard will only look for the previously connected nodes) . If the CAN node ID of any connected CNA node changes then you should recheck this control before reconnecting.

    Connecting as Virtual Nodes

    Occasionally you may want to run Drive Wizard with no vehicle connected e.g. to set up and test settings offline or to practice using Drive Wizard.

    Drive Wizard incorporates user selectable virtual CAN nodes to allow you to do this. For a full description see Using Drive Wizard in Virtual mode.

    To connect to virtual nodes you should check this control. Drive Wizard will then allow you to set up your virtual nodes by changing the connected CAN Nodes table.

    To add a virtual node you must first select a Node ID from the Drop down list in the Connected CAN nodes table. This will be the CAN node ID of the virtual CAN node. You then select the type of Node you want by selecting from the drop down list of available EDS files, product codes or manufacturers.

    To make this virtual CAN node a bus master you should check the Master column.

    You can add up to eight virtual cAN nodes.

    Testing the Bus Connection

    To connect either real or virtual CNA nodes you should click the Text Connection button. (in virtual mode Drive Wizard will 'find' the nodes that you have specified)

    When connected to a real CNA bus Drive Wizard will search the CNA bus using the criteria you set in this Baud rate controls. Drive Wizard will then display details about the the CNA nodes it has found in the Connected nodes table. Additional feedback is supplied in the status bar

    You can use either real or virtual nodes. It is not possible to use a mixture. When using virtual CAN nodes Drive Wizard should not be connected to a CAN bus.

    Closing Options screen

    This control closes the window. If you have tested the connection then Drive Wizard will rebuild the System Tree shown in the main window Structure Pane
