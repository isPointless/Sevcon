Drive Wizard Layout: Overview

Drive Wizard main view is divided into two main sections:

  • The Structure Pane which is located on the left side of the main screen
  • The Detail Pane which is located on the right side of the main screen

These panes are separated by an moveable bar allowing you to see more or less of each pane.

Drive Wizard uses separate windows for functionality that needs to be viewed concurrently with the the Detail Pane such as graphing data.

Separate windows are also used for functionality which is not part of normal System setup, monitoring and adjustment such as fine tuning a Sevcon controller to the individual characteristics of the attached motor(s).

The Structure Pane

The structure pane is used to display the connected CAN nodes and a structured view of the parameters contained in their object dictionaries. The structure view also displays icons for Drive Wizard guided processes that can be applied to either the vehicle or to individual nodes. For a definition of each icon type see System Structure icons, Guided processes can be displayed by clicking their icon. If a guided process is not currently applicable e.g. a process requires the node to be in the pre-operational state the Drive Wizard automatically removes the icon from the structure view and redisplays it once the CAN node is set to the pre-operational state. Icons representing parameters contained in a CAN nodes Object Dictionary can be clicked to view further infromation about this section of the Object Dictionary in the detail pane. Alternatively Object Dictionary items can be dragged to another destination e.g. a list of items to be graphed/monitored in real time.Right clicking in the Structure pane displays a context menu applicable to the currently highlighted Structure Pane icon e.g. Right clicking on any icon under the DCF icon will display the DCF context menu items currently applicable.

The Detail Pane

The detail panel is used primarily to display detailed infromation about Object Dictionary parameters. You can monitor individual parameters sin real time using the e detail Pane context menu. You can also change the value of individual parameters that ar available in the current context if necessary.

Changing individual parameters can affect vehicle perfromance and safety and should only be attempted by expert users. Use Drive Wizard guided processes to change the vehicle set up.
