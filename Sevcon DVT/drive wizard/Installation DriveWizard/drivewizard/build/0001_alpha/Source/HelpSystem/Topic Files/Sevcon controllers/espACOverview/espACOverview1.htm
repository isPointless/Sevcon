espAC Controllers Introduction
 Sevcon espAC controllers are designed to control 3-phase AC induction motors in battery powered traction and pump applications. A range of models is available to suit a wide number of applications and cooling regimes. For optimum perfromance and ease of installation, we offer versions with integral fan cooling

The controller adapts its output current to suit the loaing conditions and the ambient in which it is operating (temporarily shutting down if necessary). It will also protect itself if incorrectly wired.

Signal wiring and power connections have been designed to be as simple and straight forward as possible. Analog and digital signal inputs and outputs are provided for switches, sensors, contactors, hydraulic valves and CAN communications. These electrical signals can be mapped to espAC’s software functions to suit a wide range of traction and pump applications.

Given espAC’s mapping versatility it is important to ensure you map your application signals to the correct software functions (see ‘Object mapping’ on page 6-19). A common configuration is supplied by default which may suit your needs or act as a starting point for further configuration.

Configuration and control of espAC is fully customizable using Sevcon’s DriveWizard, an intuitive Windows based configuration software tool.

A single green LED is provided to give a visual indication of the state of the controller. This signal can be replicated on a dashboard mounted light for example.
