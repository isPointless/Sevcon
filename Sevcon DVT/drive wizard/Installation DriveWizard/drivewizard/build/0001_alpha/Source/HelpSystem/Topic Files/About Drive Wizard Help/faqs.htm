Guided Processes

The icon for the guided process I want to use is not visible
Drive Wizard allows me to open a guided Process but prevents me from changing any values

Drive Wizard displays the icons and or limits access to Guided Processes dependent on the current system configuration and the users login level.
If you were expecting to see a Guided Process icon then you should check the following:

Do I have enough access to perfrom this Guided Process?
Some processes require a minimum Sevcon controller login level. This Sevcon is done automatically by Drive Wizard when you Click the Connect button.in the Select Profiles Screen. If in doubt check your system access in the Main Window Status bar and re-connect using the Options screen Vehicle Profiles Tab

Is this Guided Process applicable to this CANopen node?
Some processes are only applicable to Sevcon controllers running application software. Look at the CAN node icon in the Structure Pane
Is this controller running Boot Loader or Self Characterization software? If so, it may need to be re-programmed.

Does the CAN node need to be the CAN bus master.
A CAN node that is a bus master is indicated by a red outline on its icon. All other CAN nodes have a gray outline

Is this guided Process applicable to this type of CAN node.
Some processes e.g. self characterization are only applicable to a motor controller and would not be displayed e.g. for a pump controller

Does this Guided Process require the system to be in pre-operational state?
Some processes require all connected nodes to be in the pre-operational state. Try using the toolbar icon to request the CNA bus master to force the system to pre-operational.

Is anther Guided Process already running ?
Some Guided Processes either change the System Structure and/or should not be interrupted e.g. reprogramming CAN nodes. Drive Wizard will prevent you from running other processes while these processes are running.

Structure Pane View

Drive Wizard has found my CAN node but is not showing any infromation under the CAN node icon

A CAN nodes Object Dictionary can only be displayed if Drive Wizard can access a matching EDS for the CAN node. If this is a Sevcon CAN node then click on the CAN Node icon in the Structure Panel record the CiA Vendor ID, the Product Code and the revision number. The contact the supplier of the CAN Node and quote these values. For Sevcon CAN Nodes contact the Drive Wizard Team

Drive Wizard is showing the wrong Object Dictionary structure under my CAN node icon

The EDS provides infromation on every Object Dictionary parameter for a matching CAN node. The EDS is a flat structure and therefore with large Object Dictionaries it can be difficult to locate a particular object. To overcome this, Drive Wizard applies an XML based tree structure to CAN nodes that have an associated EDS.
Sevcon controllers have detailed XML structures based on the function of each Object Dictionary object. This infromation is not available for non-Sevcon CAN nodes and so Drive Wizard applies an XML structure based on the generic CANopen Object Dictionary structure. If you see a generic structure under a Sevcon node icon then the most likely reason is that an XML file has been deleted from your system. Contact the Drive Wizard Team for a replacement quoting the CiA Vendor ID , product code and revision number of the CAN node.

Drive Wizard prevents me from dragging an icon in the System Structure

When you try to drag an icon Drive Wizard checks whether any suitable destinations exist for the current system (this is dependent on the vehicle mode, other active Drive Wizard processes and the user login level). If no suitable destinations exist then Drive Wizard will inhibit Drag&Drop for this icon. Drive Wizard will display the inhibit reason in the Status bar.
Attempting to Drag an icon too quickly can occasionally prevent Drive Wizard from seeing the Drag&Drop Request. If Drive Wizard does not display the Drag&Drop inhibit explanation in the status bar then try Clicking and Dragging slightly slower.

Drive Wizard prevents me from dropping an icon onto my destination

When you drag an icon over a potential drop site Drive Wizard will evaluate when the this is a suitable destination. this is dependent on the vehicle modes, other active Drive Wizard processes and the user login level). If this is not a suitable destination then Drive Wizard will display the inhibit reason in the status bar.
