CAN Node status view

The CAN Node Status view is displayed whenever a CAN Node Icon is highlighted in the Structure Pane the CAN Node Status view shows high level infromation about the CNA node in an easily accessible way. The CNA NODe status screen shown below is for a SEvcon controller. Less infromation is available for Non-Sevcon CNA nodes.
