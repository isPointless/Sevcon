The first time you run Drive Wizard

Drive Wizard allows you to quickly connect to and switch between different vehicle configurations.
When you first run Drive Wizard you will be presented with the Connection Options screen. This is where you tell Drive Wizard about the vehicle systems you will be connecting to by setting up a Vehicle Profile for each of your vehicle types.
Remember you can change and add new Vehicle Profiles at any time using this screen. Once you have set up one or more Vehicle Profiles then, next time you run Drive Wizard, you can just choose a Vehicle Profile and Connect.
