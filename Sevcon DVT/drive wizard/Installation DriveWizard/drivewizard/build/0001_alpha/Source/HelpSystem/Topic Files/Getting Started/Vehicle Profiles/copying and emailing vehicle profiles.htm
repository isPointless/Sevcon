Copying and Emailing Vehicle Profiles

It is not normally necessary to access Vehicle Profile files directly. If you reInstall Drive Wizard then you can still use your current veiclh profiels.

If you wish to use a Vehicle Proifle on a different PC then the profile files are located under the application data directory for your user name via the following sub path:

//Sevcon/Drive Wizard/profiles

The application data directory path can vary but is commonly found at:

<C:>/documents and settings/Documents and Settings/<user name>/Application Data

Vehicle Profile files have the extension .xml.

You should not edit a Vehicle Profile file directly as this can prevent Drive Wizard from being able to read it later.
