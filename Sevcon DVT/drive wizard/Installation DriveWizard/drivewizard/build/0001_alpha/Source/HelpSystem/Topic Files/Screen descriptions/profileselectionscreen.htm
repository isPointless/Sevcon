Profile Selection Screen

The Select Vehicle Profile screen is displayed whenever you start Drive Wizard and you have create done or more Vehicle Profiles.
Once you successfully connect to the CAN bus, Drive Wizard automatically closes this screen.
If Drive Wizard subsequently detects that one or more connected CAN nodes may have changed either their Object Dictionary e.g. if a node has been reprogrammed or that a Sevcon node login may have changed e.g. if a CAN node has restarted the Drive Wizard will attmept to stop any current processes and redisplay the Select Vehicle Profile screen to prompt you to reconnect to the CAN bus.

Select Profile drop down list

Drive Wizard displays the your default Vehicle Profile (you can change this in the Options Screen) You can choose a different Vehicle Profile to connect with from the drop down list.

Connect button

When this button os clicked Drive Wizard will attempt to connect to the CNA bus using the selected Vehicle Profile. If you have opted to use virtual nodes then Drive Wizard will set up a virtual system using the CAN nodes found when this Vehicle Profile was last used. If you successfully connect to the bus then Drive Wizard will automatically close the Select Profiles screen and set up the System structure in the Main Window Structure pane. Structure Pane

You may need to wait a few seconds for Drive Wizard to complete setting up and displaying the System structure

If Drive Wizard is unable to find any connected CAN nodes then the Select Profiles Screen will remain open for you to check the physical bus connections and vehicle power and then to retry connection. Alternatively you can edit the Vehicle Profile.

edit Vehicle Profiles button

Use this button to switch to the Options Screen to add, delete and edit your Vehicle Profiles

work Offline button

Limited Drive Wizard functionality is available with no node s(real or virtual) connected e.g. viewing a previously stored graph. Clicking this button will Close the select Profiles screen.

Use virtual nodes checkbox

When this is checked Drive Wizard will set up a virtual System Structure using the CAN nodes found last time you connected using this Vehicle Profile. If it is unchecked the Drive Wizard will assume that you are connected to a real CAN bus.

user feedback

The user instructions label displays the actions that Drive Wizard is expecting you to do.
Searching the CNA bus can take some time depening on the connection settings and the time that Drive Wizard waits for CAN nodes to respond. The connection progress bar provides an indication o the search progress.
The status bar displays Drive Wizard current activity or the results of Drive Wizards last action as appropriate e.g. if a search has completed and no CNA nodes have been found then Drive Wizard reports this here.
